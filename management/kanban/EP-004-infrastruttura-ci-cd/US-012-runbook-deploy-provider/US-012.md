# US-012 — Runbook deploy per ogni provider

## Descrizione

Come **developer**, voglio avere runbook operativi che documentino il processo di deploy per Vercel, Render e Oracle ARM, riducendo il bus factor e accelerando il troubleshooting.

## Criteri di accettazione

- [ ] Runbook Vercel: step deploy, env vars, rollback, custom domain.
- [ ] Runbook Render: step deploy, env vars, health check, cold start mitigation.
- [ ] Runbook Oracle ARM: provisioning, Docker compose, rete, storage.
