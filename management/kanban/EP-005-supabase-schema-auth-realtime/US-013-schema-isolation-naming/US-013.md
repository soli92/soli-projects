# US-013 — Schema isolation e naming convention

## Descrizione

Come **developer**, voglio che le naming convention per tabelle Supabase siano documentate e rispettate, evitando conflitti quando più progetti condividono lo stesso istanza.

## Criteri di accettazione

- [ ] Mappatura tabelle per progetto documentata (namespace usato, tabelle esistenti).
- [ ] Audit conflitti: nessuna tabella con nome ambiguo tra progetti.
- [ ] Migration plan se esistono tabelle non conformi.
