# US-005 — Integrazione submodule nei repository verticali

## Descrizione

Come **developer**, voglio che soli-projects sia incluso come git submodule nei repository verticali pilota, così che ogni repo possa leggere wiki/ e management/ direttamente e scrivere directives/.

## Criteri di accettazione

- [x] Submodule configurato in almeno 3 repo pilota (soli-agent, solids, soli-prof).
- [x] Documentazione workflow `git submodule update --remote` per sviluppatori.
- [x] CI dei repo pilota non fallisce per presenza submodule.
